Amortization Calculator

Personal project to create an amortization plan. Instead of inputting time to get the monthly payment like every online calculator, my calculator inputs maximum monthly payment and outputs total time: best suited for those who want to pay off their loan early.

Hotel Database

The final project for Database Design course, which showed relationships of the entities (customer, employee, room, bookings, transactions, etc.) and analyzed data to produce meaningful queries, forms, and reports about a hotel using SQL and Microsoft Access.

Voice Recognition

The final homework for the Data Science course, which uses labeled data of voices to train models, by using artificial neural networks from the Keras library in python. This model is then used to make predictions on voices of unknown sources.

Stock Market Predictions

The final project for the Data Science course, which analyzed past stock prices to make predictions for future stock prices, using Long Short-Term Memory (LSTM) recurrent neural networks and linear regression. This model is accurate for stable stocks, but can not predict sudden changes.
